Reporte de laboratorio N°1 
Creacion de primera maquina virtual 
Objeto: el objetivo de este laboratorio fue crear una maquina virtual para disponer de una entorno seguro e independiente donde realizar practicas, sin afectar el sistema operativo del equipo principal. 
Herramientas urilizadas:
Oracle virtual box, Imagen ISO de ubuntu desktop 
computadora windows 11
github.com 
Desarollo del laboratorio: 
1) primero se confirmo que la virtualizacion este activada en el equipo.
2) se descargo Oracle VirtualBox como sofware de virtualizacion para crear y administrar maquinas virtuales.
3) descarga del sistema operativo se descargó la imagen ISO de Ubuntu Desktop desde su sitio oficial para utilizarla como sistema operativo invitado dentro de la máquina virtual.
4)Creación de la máquina virtual :Durante la creación se configuraron los siguientes parámetros:
Nombre: Ubuntu
Sistema Operativo: Ubuntu (Linux)
Memoria RAM: (2048MB)
Procesadores: (1CPU)
Disco duro virtual: Se creó un nuevo disco virtual dinámico con capacidad suficiente para instalar Ubuntu y realizar futuras prácticas.
configuracion de red 
Elegi el modo red interna porque permite que la maquina virtual quede aislada de internet y de la red de mi computadora, de esta manera el laboratorio se desarolla en un entorno controlado y seguro, reduciendo el riesgo de afectar otros equipos o de esponer a la maquina virtual a amenazas internas.
